Глава 1 Введение в HTML

HTMl (Hyper Text Markup Language) - язык разметки с помощью которого создаются веб-страницы.
Он определяет структуру и содеожимое сайта: заголовки, текст, изображения, ссылки и прочее.
Структура HTML-документа выглядит так: 

                    
                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Document</title>
                        </head>
                        <body>
                            
                        </body>
                        </html>
Давайте разберем по строчно
Каждый HTML-документ начинается с декларации <!DOCTYPE html>, которая сообщает браузеру, что используется HTML
Далее идут корневые теги <html>, <head> и <body>.
Теги - это элименты кода которые находятся <..> и </..>
Они бывают "закрытими" то есть имеют </..> и "открытими" теги которые не надо закрывать
Закрытые теги - В коде выше это теги <html>, <head>, <body>, <title> они у нас закрытые
Открытые теги - к примеру тег <br>(перенос строки)
<html> - корневой элимент HTML документа , заключает в себе все другие элименты.
<head> - содержит мета данные о документе такие как заголовок, стили, скрипты. Тег <meta> это и есть те самые данные как видно выше
<body> - это основной контент страницы, то что мы видим в браузере
<title> - это заголовок нашей страницы который видно на вкладке(заголовке) сайта
Отлично теперь давайте создаим свою первую страницу>
Создаем на рабочем столе (или где вам удобнее) текстовый документ
ПКМ>Создать>Текстовый документ
название документа может быть любым, но расширение файла вместо .txt меняем на .html
Я создал index.html (обычно главную страницу сайта называют index)
теперь откроем этот файл в блокноте
ПКМ(по файлу)>Открыть с помощью>Блокнот
и напишем такой код: 
                    
                        <!DOCTYPE html>
                        <html lang="ru"> // lang="en" это указывает на то что основной язык документа это Английский мы поставим на ru - Русский
                        <head>
                            <meta charset="UTF-8"> // Про это нам пока рано знать :)
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Моя первая страница!</title>
                        </head>
                        <body>
                            <p>Привет мир!</p>
                        </body>
                        </html>
можно заметить новый тег <p>
